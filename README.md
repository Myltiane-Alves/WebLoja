## Criação de um Ecommerce em React.JS

Tecnologias utilizadas:
    "vite":,
    "axios":,
    "react-hook-form":,
    "react-icons":,
    "react-router-dom":,
    "react-toastify":,
    "styled-components":,
    "swiper": "^8.4.4"
    "typescript"

    **Requisito Funcional = RF**
    Deve ser Possível cadastrar um usuário
    Deve ser Possível listar as categorias
    Deve ser Possível Comprar um produto
    Deve ser Possível listar os produtos
    Deve ser Possível Favoritar um produto
    Deve ser Possível listar os produtos favoritos


    **Requisito Não Funcional = RNF**
    Não deve ser possível cadastrar um usuário com o mesmo email
    Não deve ser Possível comprar um produto sem estar logado
    Não deve ser Possível comprar um produto sem ter no estoque

    **Regras de Negócio = RN**
